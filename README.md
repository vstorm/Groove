##Groove 豆瓣FM客户端
使用python3+pyqt编写的豆瓣FM客户端

##依赖
* python3

* python3-bs4

* python3-PyQt4

##截图
<img src="http://i1299.photobucket.com/albums/ag72/vstormli/5DE54F5C533A1_011_zps1e2cfda7.png" />

##运行
>在ubuntu 12.04 Gnome和Kde环境下都可运行

>下载后运行

>`chmod +x Groove`

>然后运行Groove就可以启动程序
